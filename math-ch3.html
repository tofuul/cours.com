📘 Chapitre 3 : Suites numériques 1/2

Notions importantes

  • Suite : fonction u : N → R, notation u_n
  • Suite arithmétique : u_{n+1} = u_n + r
  • Suite géométrique : u_{n+1} = u_n × q
  • Formules : u_n = u_0 + n·r (arithmétique), u_n = u_0 × q^n (géométrique)
  • Monotone : croissante si u_{n+1} ≥ u_n, décroissante sinon
