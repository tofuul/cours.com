Chapitre 7

Titrages colorimétriques

🧪 Titrage : principe

Déterminer la concentration inconnue d’une solution (réactif titré) grâce à une solution titrante de concentration connue. Réaction type : aA + bB → cC + dD.

🔧 Montage expérimental

  • Burette : solution titrante
  • Erlenmeyer : solution titrée
  • Agitateur magnétique
  • Indicateur coloré pour repérer l’équivalence

⚖️ Équivalence

L’équivalence est atteinte lorsque les réactifs sont introduits dans les proportions stœchiométriques : n(A)/a = n(B)/b. Avant équivalence : A en excès. Après équivalence : B en excès.

👁️ Repérer l’équivalence

Changement de couleur : si titré coloré → disparition, si titrant coloré → apparition, si indicateur → virage de couleur.

🧮 Relation à l’équivalence

À connaître : aCAVA = bCBVeq. Donc : CA = b CB Veq / (a VA). Attention aux unités (L).

📈 Courbe de titrage

Quantité de titré ↓ jusqu’à 0 à l’équivalence. Quantité de titrant ↑ après équivalence. Point clé : cassure à l’équivalence.

🚀 Méthode type

  1. Écrire l’équation chimique.
  2. Identifier titré / titrant.
  3. Repérer Veq.
  4. Utiliser : aCAVA = bCBVeq
  5. Calculer CA.
