📘 Chapitre 1 : Fonctions polynômes du second degré

Notions importantes

  • Forme générale : f(x) = ax² + bx + c
  • Discriminant : Δ = b² - 4ac
  • Racines : x = (-b ± √Δ) / 2a
  • Sommet : S(-b/2a ; -Δ/4a)
  • Variations selon a et Δ

Formules utiles

  • Δ = b² - 4ac
  • x₁ = (-b - √Δ)/2a, x₂ = (-b + √Δ)/2a
  • Sommet : S(x_s ; y_s) = (-b/2a ; -Δ/4a)
  • Forme factorisée : a(x - x₁)(x - x₂) si Δ ≥ 0
